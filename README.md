This project is a frontend simulation of the SAP procurement process. It demonstrates the basic purchasing workflow including Purchase Requisition (PR), Purchase Order (PO), and Goods Receipt (GR).

## Features
- Purchase Requisition (PR) creation
- Convert PR to Purchase Order (PO)
- Goods Receipt (GR) process
- Company and Vendor login dashboards
- Order status tracking
- Simple messaging between company and vendor
- Modern and responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
The goal of this project is to help students understand the SAP S/4HANA sourcing and procurement workflow through an interactive frontend demonstration without requiring an actual SAP backend system.
